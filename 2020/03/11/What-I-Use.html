Operating System

On my main desktop machine I have Windows 10 dual booted with Arch Linux and just Arch Linux on my Laptop. I am planning to switch to Linux completely. However, some software I use for my University are Windows exclusive.

Music

I use ncmcpp as my music client together with Mopidy.

Mopidy has an amazing collection of plugins where you can basically combine music from Youtube, Spotify and Soundcloud into one central music player. I don’t like saving music to my hard-drive anymore so this basically perfect.

Text Editor

I use vim for prettty much anything. In conjuction with plugins like CoC it’s basically a fully fledged IDE, but runs fast.

Browser

qutebrowser for most of my browsing needs. The browser has vim keybindings and a lot of customisation options with a configuration file. I don’t personally have any crazy shortcuts set but it’s there.

Document Writing

I use LateX for all of my academic papers. There is a learning curve but if you are still using Microsoft Word, I suggest to give it a try.

Music Production

FL Studio is the program I’m most familiar and basically, enjoy. Sadly, there are no sensible Linux alternatives for music production, yet.

Image Editing

Gimp is perfect for basic to advanced photo editing.

Font

My favorite font has to be Cozette. It looks great on any context, plus it includes glyphs compatible with ranger-icons. Most people will tell you bitmap fonts are only good for ricing but they are definetely wrong. Let them keep using Source Code Pro.

Terminal Emulator

Alacritty is great. It works out of the box and has a simple, readable config file.

Window Manager

i3 is the first and last window manager I’ve installed.
